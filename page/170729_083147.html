    • remoteStorage.

      • remotestorage.io : An open protocol for per-user storage on the Web.
      • 5apps.com : A remoteStorage implement service, much better than other cloud solution when consider privacy. In the storage page, you can chose to revoke the permission of an app anytime.

      • laverna.cc : Keep your note private. Secure client-side encryption (end-to-end). Synchronizes with cloud storage services (currently only with Dropbox and RemoteStorage). No registration required. Web based. Markdown editor based on Pagedown.

        When using laverna, it seems to save data in your html5 local storage, not on cloud. The way you use cloud and sync your data between phone and PC is, in both side, login to remoteStorage service for instance 5apps.com, by this the data will save on it's remoteStorage server and can be seen by both side.

        Aware if you use encryption, you have to set it in both side! Or the data will not able to be seen in other side.

        Also see github for more infor.
