* Simple example,, by me.

The basic setup is include <sjcl.js> and <sha512.js> (if you want to use sha512) script first:

<script src="./sjcl-master/sjcl.js" type="text/javascript"></script>
<script src="./sjcl-master/core/sha512.js" type="text/javascript"></script>

Then you can use js code like this:

var out = sjcl.hash.sha512.hash("foobar");
var hash = sjcl.codec.base64.fromBits(out)
console.log( hash );
